# GPT Pear SVG Demo

This demo uses a GPT-generated transparent PNG as the source image for the SVG route.

## Files

- `source.png`: transparent PNG source.
- `pear.svg`: vtracer output generated by `png2svg.py`.
- `idle.svg.html`: runnable idle animation using the generated SVG.

## Source Check

- Image size: 1254 x 1254
- Transparency: real RGBA alpha, transparent corners
- Visible alpha: hard edge only, no semi-transparent pixels
- Visible RGB colors before quantization: about 16k

## Reproduce

From the repository root:

```powershell
py -3.13 routes\svg\tools\png2svg\png2svg.py examples\svg-gpt-pear\source.png examples\svg-gpt-pear\pear.svg --preset apple-precise
```

This run produced 13 SVG paths and a 21 KB SVG file.

## Result Notes

This is a better source for vtracer than a screenshot with a checkerboard background. The true alpha channel prevents the background from being vectorized, and the 8-color quantization keeps path count low.

The source still contains soft shading, so the SVG output is an approximation of the original image rather than a hand-clean vector drawing. For production-quality SVG pets, use flatter colors or redraw the main shapes manually after tracing.
