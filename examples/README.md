# Examples

Runnable demos and public-safe case-study notes.

## Runnable Demos

- `svg-gpt-pear/`: primary README demo; GPT-generated transparent PNG to vtracer SVG to idle animation. Also holds `pear-gpt-5.5-pro.svg`, the same character authored directly by GPT-5.5 Pro, for the tool-trace vs direct-model-authoring comparison in the README.
- `svg-soft-orb/`: synthetic baseline demo from a hand-made low-color PNG to vtracer SVG to idle animation.

## Notes

- `svg-case-study-notes.md`: reusable SVG route lessons without private assets.
- `apng-runtime-notes.md`: reusable APNG/runtime lessons without private assets.
